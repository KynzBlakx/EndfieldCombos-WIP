Simulador de Combos para Arknights: Endfield

Estou desenvolvendo um site que busca simular os combos entre os personagens para facilitar a criação e otimização de equipes.

O projeto ainda está em estágio inicial e a lógica do sistema continua em desenvolvimento. Ainda não posso garantir que os resultados sejam totalmente precisos, pois muitas mecânicas ainda estão sendo implementadas e refinadas.

Há bastante trabalho pela frente, além de muitas ideias que pretendo adicionar ao longo do desenvolvimento. O objetivo é criar uma ferramenta completa, capaz de compreender as habilidades, sinergias e interações entre os personagens, oferecendo uma forma prática de montar equipes e explorar diferentes estratégias.

Arknights: Endfield Combo Simulator

I'm working on a website that aims to simulate character combos to help players build and optimize their teams.

The project is still in its early stages, and the core logic is actively being developed. I can't guarantee that the results are fully accurate yet, as many mechanics are still being implemented and refined.

There's still a long way to go, and I have plenty of ideas I'd like to add over time. My goal is to create a comprehensive tool capable of understanding each character's abilities, synergies, and interactions, making it easier to experiment with team compositions and discover new strategies.# EndfieldCombos
